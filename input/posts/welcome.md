Welcome to the **Phantom** theme for Statiq!

This is an example blog post. You can add your own posts to the `input/posts` folder.

## Getting Started

Add Markdown files to `input/posts/` with YAML front matter, like this:

```markdown
Title: My Post Title
Lead: A brief description
Published: 2024-01-01
Tags:
  - Example
---
Your post content here.
```

## Features

- Phantom visual design from [HTML5 UP](https://html5up.net/phantom)
- Sidebar navigation menu
- Tag-based post organization
- RSS and Atom feeds
- Optional search functionality
- Optional comments via Giscus
